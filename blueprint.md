# 프로젝트 블루프린트: 학생 이수 학점 점검 도구

## 1. 개요

본 프로젝트는 고등학교 학생들의 교과목별 이수 학점을 자동으로 분석하고, 필수 이수 학점 기준 충족 여부를 손쉽게 확인하기 위한 웹 애플리케이션입니다. 교사는 엑셀 파일 형태로 된 학생 성적 데이터를 업로드하여 개별 학생의 교과 영역별 이수 현황을 한눈에 파악할 수 있습니다.

## 2. 핵심 기능 및 설계

### 2.1. 필수 이수 학점 기준 설정

- **UI**: 사용자는 웹 페이지에서 직접 각 교과 영역의 최소 이수 학점 기준을 설정할 수 있습니다.
- **설정 항목 (v2.0 기준)**:
  - 국어, 수학, 영어, 한국사, 사회, 과학, 체육, 예술, 생활/교양

### 2.2. 엑셀 데이터 분석 (v2.0)

- **파일 업로드**: 사용자는 `.xlsx` 또는 `.xls` 형식의 엑셀 파일을 선택하여 업로드합니다. 파일 선택 즉시 분석이 자동으로 시작됩니다.
- **데이터 통합 처리**: 엑셀 파일 내에 동일한 학생(같은 '번 호'와 '성 명')의 데이터가 여러 섹션에 나뉘어 있어도, 이를 **하나의 학생 데이터로 통합**하여 모든 과목 기록을 합산합니다. 학생 식별은 `번 호`와 `성 명`을 조합한 고유 키를 사용합니다.
- **교과 분류 로직**:
  - **기본 분류**: '교과' 필드를 기준으로 '국어', '수학', '영어', '과학', '사회', '체육', '예술'을 분류합니다.
  - **특수 분류**: 
    - 과목명에 '한국사'가 포함된 경우 '한국사'로 우선 분류합니다.
    - '교과' 필드에 '기술·가정', '정보', '제2외국어', '한문', '교양' 등이 포함된 경우 '생활/교양'으로 통합 분류합니다.
  - **미분류 처리**: 위의 기준에 해당하지 않는 모든 과목은 '미분류' 항목으로 별도 집계하여 결과에 표시합니다.

### 2.3. 분석 결과 표시

- **결과 테이블**: 분석이 완료되면 각 학생의 이름, 교과 영역별 충족 여부, 그리고 미분류된 과목 목록이 테이블 형태로 명확하게 표시됩니다.
  - **충족**: '충족' 상태와 함께 총 이수 학점이 표시됩니다. (예: `충족 (12)`)
  - **미충족**: '미충족' 상태와 함께 부족한 학점이 음수로 표시됩니다. (예: `미충족 (-2)`)

### 2.4. 엑셀로 내보내기

- **기능**: 사용자는 '엑셀로 내보내기' 버튼을 클릭하여 현재 화면에 표시된 분석 결과를 `.xlsx` 파일로 다운로드할 수 있습니다.
- **내용**: 다운로드된 엑셀 파일은 화면의 결과 테이블과 동일한 구조와 내용을 가집니다.

## 3. 기술 스택

- **Frontend**: HTML, CSS, JavaScript (ES Modules)
- **라이브러리**: `xlsx.js` (SheetJS) - 엑셀 파일 파싱 및 생성을 위해 CDN을 통해 사용합니다.
