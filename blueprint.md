# 프로젝트 블루프린트: 학생 이수 학점 점검 도구

## 1. 개요

본 프로젝트는 고등학교 학생들의 교과목별 이수 학점을 자동으로 분석하고, 필수 이수 학점 기준 충족 여부를 손쉽게 확인하기 위한 웹 애플리케이션입니다. 교사는 엑셀 파일 형태로 된 학생 성적 데이터를 업로드하여 개별 학생의 교과 영역별 이수 현황을 한눈에 파악할 수 있습니다.

## 2. 핵심 기능 및 설계

### 2.1. 필수 이수 학점 기준 설정

- **UI**: 사용자는 웹 페이지에서 직접 각 교과 영역의 최소 이수 학점 기준을 설정할 수 있습니다.
- **설정 항목 (v2.1 기준)**:
  - 국어, 수학, 영어, 한국사, 사회, 과학, 체육, 예술, 생활/교양

### 2.2. 엑셀 데이터 분석 (v2.1)

- **파일 업로드**: 사용자는 `.xlsx` 또는 `.xls` 형식의 엑셀 파일을 선택하여 업로드합니다. 파일 선택 즉시 분석이 자동으로 시작됩니다.
- **데이터 파싱**: 엑셀 데이터의 각 행을 읽어 학생별 과목 이수 정보를 추출합니다.
- **동적 헤더 감지**: '성명', '교과', '학점' 등의 헤더가 포함된 행을 동적으로 감지하여, 여러 다른 양식의 엑셀 파일에 유연하게 대응합니다.
- **교과 분류 로직**:
  - **기본 분류**: '교과' 필드를 기준으로 '국어', '수학', '영어', '과학', '사회', '체육', '예술'을 분류합니다.
  - **특수 분류**: 
    - 과목명에 '한국사'가 포함된 경우 '한국사'로 우선 분류합니다.
    - '교과' 필드에 '기술·가정', '정보', '제2외국어', '한문', '교양' 등이 포함된 경우 '생활/교양'으로 통합 분류합니다.
  - **미분류 과목 처리**: 위의 기준에 해당하지 않는 과목은 계산에서 제외되며, 별도의 보고 섹션에 목록화됩니다.

### 2.3. 분석 결과 표시

- **결과 테이블**: 분석이 완료되면 각 학생의 이름, 교과 영역별 충족 여부, 그리고 '국영수합'이 테이블 형태로 명확하게 표시됩니다.
  - **충족**: '충족' 상태와 함께 총 이수 학점이 표시됩니다. (예: `충족 (12)`)
  - **미충족**: '미충족' 상태와 함께 부족한 학점이 음수로 표시됩니다. (예: `미충족 (-2)`)
  - **국영수합**: 학생별 국어, 수학, 영어 이수 학점의 총합과 전체 기준 학점(174점) 대비 비율을 함께 표시합니다.

- **미분류 과목 보고 (v2.1 신규)**:
  - **조건부 표시**: 분석 과정에서 분류되지 않은 과목이 발견될 경우에만, 결과 테이블 하단에 별도의 보고 섹션이 나타납니다.
  - **상세 정보**: 섹션의 제목은 `미분류 과목 (n개)`와 같이 총 개수를 표시합니다.
  - **내용**: 각 미분류 과목은 `학생명: 과목명(원본 교과)` 형식으로 목록화되어, 어떤 학생의 어떤 과목이 왜 집계에서 누락되었는지 쉽게 파악할 수 있습니다.

### 2.4. 엑셀로 내보내기

- **기능**: 사용자는 '엑셀로 내보내기' 버튼을 클릭하여 현재 화면에 표시된 분석 결과를 `.xlsx` 파일로 다운로드할 수 있습니다.
- **내용**: 다운로드된 엑셀 파일은 화면의 결과 테이블과 동일한 구조와 내용을 가집니다.

## 3. 기술 스택

- **Frontend**: HTML, CSS, JavaScript
- **라이브러리**: `xlsx.js` (SheetJS) - 엑셀 파일 파싱 및 생성을 위해 CDN을 통해 사용합니다.

## 4. 버전별 변경 로그

### v2.0
- **주요 업데이트**: 엑셀 파일 내 분리된 학생 데이터를 통합하여 분석하는 기능 추가.
- **기능 추가**: '체육', '예술' 교과 영역 추가 및 '국영수합' 열 추가.
- 동적 헤더 감지 기능 도입으로 파싱 안정성 향상.

### v2.1 (현재 버전)
- **신규 기능**: 분류되지 않은 과목을 결과 하단에 별도로 보고하는 기능 추가. (`과목명(원본교과)` 형식)
- **개선**: 국영수합 계산 기준을 192점에서 174점으로 변경.
- **개선**: 전체 학생 평균 국/수/영 총계 표시 부분 제거하여 UI 간소화.
